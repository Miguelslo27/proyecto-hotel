Etiquetas HTML

Títulos

  • h1 - Título de nivel 1 (Heading level 1)
  • h2 - Título de nivel 2
  • h3 - Título de nivel 3
  • ... hasta el h6

Ejemplos:

Título de nivel 1

Título de nivel 2

Título de nivel 3

Título de nivel 4

Título de nivel 5
Título de nivel 6

Párrafo

El párrafo se define con la etiqueta p. (Paragraph)

Listas

  • UL : Listas Desordenadas (Unordered List)
  • OL : Listas ordenadas (Ordered List)
  • DL : Listas de descripción (Description List)

Ejemplos:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
Elemento 1 (Description Title)
Descripción del elemento 1 (Description Description)
Elemento 2
Descripción del elemento 2
Elemento 3
Descripción del elemento 3
